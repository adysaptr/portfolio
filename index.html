Online Shop

Web-based online shop built using Bootstrap for the frontend and MySQL for the local database. The system allows users to browse and manage product listings. This project taught me the basics of front-end to back-end connectivity in web applications.

Landing Pages Web

Responsive landing page created using Bootstrap and JavaScript, designed for portfolios or product promotion. Emphasizes clean layout and user-friendly UI. I learned how to structure visually appealing and responsive web pages.

Analog Line Follower

Analog-based line follower robot using a photodiode sensor and comparator circuit to follow track lines. Designed for basic robotics learning and sensor integration. This helped me understand analog sensor behavior and control logic in embedded systems.

Android Software

Android application developed using Android Studio and Java. This project demonstrates mobile development skills and UI/UX design for real-time user interaction. It helped me practice building structured layouts and managing user input in Android apps.

IoT With ESP32

IoT system using the ESP32 microcontroller, allowing real-time remote monitoring over the internet. Ideal for smart home or environmental monitoring applications. I gained experience handling connectivity and data transmission in low-power IoT systems.

Airframe Innovation

UAV airframe system designed with a focus on structural strength, weight efficiency, and stable flight performance. The project involved integrating flight controllers, sensors, and actuators, as well as programming autonomous flight behavior on an embedded platform.

Mobile Face Recognition

Face recognition-based attendance system developed using Android Studio and the FaceNet deep learning model. Facial data is processed and stored locally using SQLite, enabling offline identification and fast check-in validation.

Object Detection

Object detection system built for an autonomous mobile robot, combining YOLOv8-seg for segmentation and LiDAR sensors for real-time distance and angle estimation. Developed using ROS for synchronized sensor communication and system control.
